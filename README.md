# Python-Anything

学习 Python ，巩固知识，探索新知。 <br />
该工程使用 **uv** 包管理工具，不使用 Python 自带的 pip 工具。<br />
该工程使用 **Jupyter Notebook** 来记录笔记。

## 更新说明（2025-12-01）

学习 Numpy 库

## ⚙️ 依赖安装

需要在本地先安装 uv，在 VSCode 中下载 Jupyter Notebook 拓展，安装过程不在这里赘述

```bash
uv sync
```
