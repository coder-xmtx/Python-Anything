# <img src="src/Python.svg" /> Python-Anything

学习 Python ，巩固知识，探索新知。 <br />
该工程使用 **uv** 包管理工具，不使用 Python 自带的 pip 工具。<br />
该工程使用 **Jupyter Notebook** 来记录笔记。

## 🛠️ 更新说明（2025-12-17）

学习 Pandas 库 - Series 概念

## ⚙️ 依赖安装

需要在本地先安装 uv 才能执行以下命令，还需要 Jupyter Notebook <br />
具体安装过程不在这里赘述

```bash
uv sync
```
